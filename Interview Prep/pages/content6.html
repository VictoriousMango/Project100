Linux Commands

  1. What are Linux Commands

    Linux commands are instructions given to a Linux operating system via a terminal or shell to perform specific tasks, such as file manipulation, system monitoring, or user management. They are executed in a command-line interface (CLI).

  2. ls (dir equivalent)

    The `ls` command lists directory contents. In Linux, there is no direct `dir` command (like in Windows), but `ls` serves a similar purpose with options like `-l` for detailed listing and `-a` to show hidden files. 

    ls -l  # Detailed list of files
ls -a  # Show all files, including hidden ones
  3. cd (Change Directory)

    The `cd` command changes the current working directory. Use `cd ..` to move up one directory, `cd /path` to go to a specific directory, or `cd ~` to return to the home directory. 

    cd /home/user  # Move to /home/user
cd ..          # Move up one level
cd ~           # Go to home directory
  4. pwd (Print Working Directory)

    The `pwd` command displays the full path of the current working directory. 

    /home/user  # Example output of pwd
  5. mkdir (Make Directory)

    The `mkdir` command creates a new directory. Use `-p` to create parent directories as needed. 

    mkdir my_folder          # Create a single directory
mkdir -p parent/child    # Create nested directories
  6. cp (Copy)

    The `cp` command copies files or directories. Use `-r` to copy directories recursively. 

    cp file.txt file_copy.txt      # Copy a file
cp -r folder/ new_folder/      # Copy a directory
  7. mv (Move/Rename)

    The `mv` command moves or renames files and directories. 

    mv oldname.txt newname.txt    # Rename a file
mv file.txt /home/user/       # Move a file to another directory
  8. rm (Remove)

    The `rm` command deletes files or directories. Use `-r` for recursive deletion and `-f` to force removal without prompting. 

    rm file.txt           # Remove a file
rm -rf folder/        # Remove a directory and its contents
  9. cat (Concatenate and Display)

    The `cat` command displays the contents of a file or concatenates multiple files. 

    cat file.txt          # Display file content
cat file1.txt file2.txt > combined.txt  # Concatenate files
  10. grep (Search Text)

    The `grep` command searches for a pattern in files or input. Use `-i` for case-insensitive search and `-r` for recursive search in directories. 

    grep "error" log.txt      # Search for "error" in log.txt
grep -r "TODO" /home/user # Recursively search for "TODO"
  1. List and Count Files

    Use Linux commands to list all files in a directory and count them. 

    ls -l | wc -l  # List files and count lines (including header)
    Output: Number of files + 1 (for header)
