Data Structures in Python

  1. What are Data Structures

    Data structures are specialized formats for organizing, storing, and managing data in a way that enables efficient access and modification. Python provides built-in data structures like lists, tuples, dictionaries, and sets, as well as support for custom implementations.

  2. Lists

    A mutable, ordered collection of items that can hold elements of different types. Lists are versatile and support operations like appending, inserting, and slicing. 

    my_list = [1, "hello", 3.14]
my_list.append(5)
print(my_list[1:3])  # Slicing
  3. Tuples

    An immutable, ordered collection of items. Tuples are similar to lists but cannot be modified after creation, making them useful for fixed data. 

    my_tuple = (1, "hello", 3.14)
print(my_tuple[1])  # Accessing element
# my_tuple[1] = "world"  # Raises TypeError
  4. Dictionaries

    A mutable, unordered collection of key-value pairs. Dictionaries provide fast lookups based on keys and are ideal for mapping relationships. 

    my_dict = {"name": "Alice", "age": 25}
my_dict["city"] = "New York"
print(my_dict["name"])
  5. Sets

    A mutable, unordered collection of unique elements. Sets are useful for operations like union, intersection, and difference. 

    my_set = {1, 2, 3, 3}  # Duplicates are removed
my_set.add(4)
print(my_set)
  6. Stacks

    A Last-In-First-Out (LIFO) data structure. In Python, stacks can be implemented using a list with `append()` for push and `pop()` for pop. 

    stack = []
stack.append(1)  # Push
stack.append(2)
print(stack.pop())  # Pop: 2
print(stack)  # [1]
  7. Queues

    A First-In-First-Out (FIFO) data structure. Python’s `collections.deque` is commonly used for efficient queue operations. 

    from collections import deque
queue = deque([1, 2, 3])
queue.append(4)  # Enqueue
print(queue.popleft())  # Dequeue: 1
print(queue)
  8. Linked Lists

    A linear data structure where elements (nodes) are linked using pointers. Python doesn’t have a built-in linked list, but it can be implemented using classes. 

    class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = new_node

ll = LinkedList()
ll.append(1)
ll.append(2)
  9. Binary Trees

    A hierarchical data structure where each node has at most two children (left and right). Binary trees can be implemented using classes in Python. 

    class Node:
    def __init__(self, value):
        self.value = value
        self.left = None
        self.right = None

root = Node(1)
root.left = Node(2)
root.right = Node(3)
