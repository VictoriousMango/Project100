SQL (Structured Query Language)

  1. What is SQL

    SQL (Structured Query Language) is a standard language for managing and manipulating relational databases. It is used to create, read, update, and delete data (CRUD operations).

  2. Tables and Schema

    A table is a collection of rows and columns storing data, defined by a schema that specifies column names and data types. 

    CREATE TABLE employees (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    salary DECIMAL(10, 2)
);
  3. SELECT Query

    The `SELECT` statement retrieves data from one or more tables, with optional filtering, sorting, and grouping. 

    SELECT name, salary 
FROM employees 
WHERE salary > 50000 
ORDER BY salary DESC;
  4. INSERT, UPDATE, DELETE

    These commands modify data in a table: `INSERT` adds rows, `UPDATE` modifies existing rows, and `DELETE` removes rows. 

    INSERT INTO employees (id, name, salary) 
VALUES (1, 'Alice', 60000);

UPDATE employees 
SET salary = 65000 
WHERE id = 1;

DELETE FROM employees 
WHERE id = 1;
  5. Joins

    Joins combine rows from multiple tables based on a related column. Types include INNER JOIN, LEFT JOIN, RIGHT JOIN, and FULL JOIN. 

    SELECT e.name, d.department_name
FROM employees e
INNER JOIN departments d
ON e.dept_id = d.id;
  6. Indexes

    Indexes improve query performance by allowing faster data retrieval, created on columns frequently used in WHERE or JOIN clauses. 

    CREATE INDEX idx_name 
ON employees (name);
  7. Aggregate Functions

    Aggregate functions (e.g., COUNT, SUM, AVG, MIN, MAX) perform calculations on a set of rows and return a single value. 

    SELECT AVG(salary), COUNT(*) 
FROM employees 
GROUP BY dept_id;
  8. Transactions

    Transactions ensure data integrity by grouping SQL statements into a single unit that either fully succeeds or fails (ACID properties: Atomicity, Consistency, Isolation, Durability). 

    BEGIN TRANSACTION;
UPDATE accounts SET balance = balance - 100 WHERE id = 1;
UPDATE accounts SET balance = balance + 100 WHERE id = 2;
COMMIT;
  9. SQL with Python

    Python integrates with SQL databases using libraries like `sqlite3` for SQLite or `pymysql` for MySQL, allowing programmatic data manipulation. 

    import sqlite3

conn = sqlite3.connect("example.db")
cursor = conn.cursor()
cursor.execute("CREATE TABLE IF NOT EXISTS users (id INT, name TEXT)")
cursor.execute("INSERT INTO users VALUES (1, 'Bob')")
conn.commit()
cursor.execute("SELECT * FROM users")
print(cursor.fetchall())
conn.close()
  1. Employee Salary Report

    Write an SQL query to find the total and average salary per department. 

    SELECT d.department_name, 
       SUM(e.salary) AS total_salary, 
       AVG(e.salary) AS avg_salary
FROM employees e
JOIN departments d ON e.dept_id = d.id
GROUP BY d.department_name;
