Computer Networks

  1. What is a Computer Network

    A computer network is a collection of interconnected devices (e.g., computers, servers, routers) that communicate to share resources and data using protocols.

  2. OSI Model

    The OSI (Open Systems Interconnection) Model is a conceptual framework with 7 layers to standardize networking functions: Physical, Data Link, Network, Transport, Session, Presentation, and Application.

    Layer Function
    7. Application User interface (e.g., HTTP)
    4. Transport Reliable data transfer (e.g., TCP)
    3. Network Routing and IP addressing
    1. Physical Hardware transmission
  3. TCP/IP Model

    The TCP/IP Model is a practical 4-layer model (Application, Transport, Internet, Link) that underpins the Internet, combining some OSI layers for simplicity.

  4. IP Addressing

    IP addresses uniquely identify devices on a network. IPv4 uses 32-bit addresses (e.g., 192.168.1.1), while IPv6 uses 128-bit addresses (e.g., 2001:0db8::1). 

    import socket
print(socket.gethostbyname("google.com"))  # Resolve IP address
  5. Protocols (TCP/UDP)

    TCP (Transmission Control Protocol) ensures reliable, ordered data delivery, while UDP (User Datagram Protocol) is faster but unreliable, used for streaming.

    Protocol Features
    TCP Reliable, connection-oriented
    UDP Fast, connectionless
  6. Socket Programming

    Sockets provide an interface for network communication between devices. Pythonâ€™s `socket` module enables TCP and UDP communication.

  7. DNS (Domain Name System)

    DNS translates human-readable domain names (e.g., google.com) into IP addresses for network communication. 

    import socket
print(socket.getaddrinfo("example.com", 80))  # DNS lookup
  8. Routing

    Routing determines the path data packets take across networks, managed by routers using protocols like RIP, OSPF, or BGP.

    (Conceptual; managed by network hardware)

  9. Network Topologies

    Network topologies describe the arrangement of devices in a network, e.g., Star, Bus, Ring, or Mesh.

    Topology Description
    Star All devices connect to a central hub
    Mesh Every device connects to every other
  1. Simple TCP Client-Server Chat

    Create a simple TCP client-server chat application using Python sockets. 

    # Server
import socket

server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server.bind(("127.0.0.1", 12345))
server.listen(1)
conn, addr = server.accept()
conn.send("Hello from server!".encode())
print(conn.recv(1024).decode())
conn.close()
                    
    # Client
import socket

client = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
client.connect(("127.0.0.1", 12345))
print(client.recv(1024).decode())
client.send("Hello from client!".encode())
client.close()
    Output: Server and client exchange messages
