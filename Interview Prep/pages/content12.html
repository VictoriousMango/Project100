MongoDB

  1. What is MongoDB

    MongoDB is a NoSQL, document-oriented database that stores data in flexible, JSON-like documents (BSON format) instead of tables. It is designed for scalability and handling unstructured data.

  2. Databases and Collections

    A MongoDB database contains collections, which are groups of documents (analogous to tables in SQL). Documents are individual records with key-value pairs. 

    use mydb;  // Switch to database
db.createCollection("users");  // Create a collection
  3. Inserting Documents

    Documents are inserted into collections using `insertOne` or `insertMany` commands. Each document can have a different structure. 

    db.users.insertOne({
    name: "Alice",
    age: 25,
    city: "New York"
});
  4. Querying Documents

    The `find` method retrieves documents from a collection, with optional filters, sorting, and projections. 

    db.users.find({ age: { $gt: 20 } })  // Find users older than 20
    .sort({ name: 1 });  // Sort by name ascending
  5. Updating Documents

    The `updateOne` or `updateMany` methods modify existing documents, using operators like `$set` or `$inc`. 

    db.users.updateOne(
    { name: "Alice" },
    { $set: { age: 26 } }
);
  6. Deleting Documents

    The `deleteOne` or `deleteMany` methods remove documents from a collection based on a query. 

    db.users.deleteOne({ name: "Alice" });
  7. Indexes

    Indexes improve query performance by creating efficient data structures for frequently queried fields. 

    db.users.createIndex({ name: 1 });  // Ascending index on name
  8. Aggregation

    The aggregation pipeline processes documents through stages like `$match`, `$group`, and `$sort` to perform complex queries. 

    db.users.aggregate([
    { $match: { age: { $gt: 20 } } },
    { $group: { _id: "$city", avgAge: { $avg: "$age" } } }
]);
  9. MongoDB with Python

    Python integrates with MongoDB using the `pymongo` library, allowing programmatic CRUD operations. 

    from pymongo import MongoClient

client = MongoClient("mongodb://localhost:27017/")
db = client["mydb"]
users = db["users"]
users.insert_one({"name": "Bob", "age": 30})
print(list(users.find({"age": {"$gt": 25}})))
