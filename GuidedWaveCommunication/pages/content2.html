Experiment 1: Modes in Optical Fiber

Aim

To study the output modes of a single-mode fiber with the change of input polarization.

Apparatus

The equipment and materials required for the experiment are:

  • He-Ne Laser Source
  • Mirrors
  • Beam Splitters
  • Wave Plates
  • Objective Lens
  • CCD (Charge-Coupled Device)
  • Single-mode Optical Fibers

Theory

Optical fibers can guide light through their core by total internal reflection. A single-mode fiber supports only one mode of light propagation, typically the fundamental mode, due to its small core diameter. The polarization of light affects the mode structure and intensity distribution at the output of the fiber.

Changing the input polarization modifies the orientation of the electric field, resulting in varying output modes that can be observed using a CCD.

Procedure

Follow these steps to perform the experiment:

  1. Set up the He-Ne laser as the light source and direct its beam using mirrors and beam splitters.
  2. Pass the beam through wave plates to control the input polarization.
  3. Focus the light into a single-mode fiber using an objective lens.
  4. Connect the output of the fiber to a CCD to observe the modes.
  5. Gradually vary the input polarization using the wave plates and note the changes in the output mode patterns.

Observations

Record the changes in output modes as the input polarization is varied. Use a table for clarity:

Input Polarization (Angle) Observed Mode Pattern
0° Fundamental Mode
45° Elliptical Mode
90° Linear Mode (Orthogonal)

Conclusion/Results

The experiment demonstrated that the output modes of a single-mode fiber depend on the input polarization. By varying the polarization angle, different mode patterns were observed, highlighting the relationship between polarization and fiber modes.
